# CEE491-591-Process-Hydrology-Tutorials
This repository contains hands-on Python tutorials covering fundamental concepts in Process Hydrology, developed for the CEE 491/591 Process Hydrology course at the University of Alabama. Topics include physical evapotranspiration (ET) modeling, physics-informed machine learning approaches for ET, and ecohydrological applications.
